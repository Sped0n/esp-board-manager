BMGR Design Principles
======================

:link_to_translation:`zh_CN:[中文]`

This chapter explains the working model of BMGR: how board-level YAML is transformed into generated code and integrated into the build, and how devices and peripherals are organized at runtime. Specific board-level field syntax is not covered here; refer to the *Board Creation Guide* and *Reference Manual* instead.
