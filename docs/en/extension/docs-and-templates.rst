Keeping Documentation and Templates in Sync
============================================

:link_to_translation:`zh_CN:[中文]`

This section explains how to synchronize template documents, reference pages, and example code after adding a new device or peripheral type.

Planned topics:

- Updating the template YAML.
- Updating the device or peripheral reference page.
- Updating minimal configuration examples.
- Updating real board configurations or test samples.
- Avoiding inconsistencies between documented fields and parser-supported fields.
