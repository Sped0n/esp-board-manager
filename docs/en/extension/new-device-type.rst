How to Add a New Device Type
==============================

:link_to_translation:`zh_CN:[中文]`

This section describes the directories to modify, the files to prepare, and how the parser and runtime implementation work together when adding a new device type.

Planned topics:

- Adding a ``devices/dev_xxx`` directory.
- Preparing the device template YAML, parser, runtime implementation, and header file.
- Explaining how the generated code calls the device implementation.
- Supplementing the device reference page, template documentation, and examples.
